# This file grandfathers existing reference / tutorial pages whose
# snippets haven't yet been extracted into runnable files. New
# tutorials and how-tos must use the `--8<--` snippet inclusion pattern
# (see scripts/check_doc_snippets.py for the rule). To migrate a page
# OFF this list:
#
#   1. Extract the inline snippet to docs/examples/<name>.<ext>
#   2. Wire the file into CI (.github/workflows/ci.yml has a
#      "Run Python example" / "Run Node.js example" pattern).
#   3. Replace the inline block with `--8<-- "examples/<name>.<ext>"`.
#   4. Remove the file from this list.
