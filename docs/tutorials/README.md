# Tutorials

Step-by-step lessons to get you productive with FLOX.

## Prerequisites

- C++20 compiler (GCC 13+ or Clang 16+)
- CMake 3.22+
- Linux (recommended)

## Getting Started

| Tutorial | Time | What You'll Learn |
|----------|------|-------------------|
| [Quickstart](quickstart.md) | 5 min | Build FLOX and run the demo |
| [First Strategy](first-strategy.md) | 15 min | Write a simple trading strategy |
| [Recording Data](recording-data.md) | 10 min | Capture live market data to disk |
| [Backtesting](backtesting.md) | 10 min | Replay recorded data through your strategy |

## Recommended Order

1. **Quickstart** — Verify your environment works
2. **First Strategy** — Understand the core programming model
3. **Recording Data** — Set up market data capture
4. **Backtesting** — Test strategies against historical data

After completing these tutorials, move on to [How-To Guides](../how-to/README.md) for specific tasks or [Explanation](../explanation/README.md) for deeper understanding.
